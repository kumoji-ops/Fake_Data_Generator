# Fake Data Generator

A simple tool that creates fake names, emails, and phone numbers for testing.

## What it does
- Generates random names (22,000+ combinations)
- Creates emails with Gmail, Yahoo, Hotmail, or custom domains
- Makes phone numbers with country codes from around the world
- Pick how many you want (1 to 100+)

## Why use it
- Test your apps with realistic data
- Fill databases quickly
- No signup, no cost, works right in your browser

## How it works
1. Open the page
2. Choose how many records
3. Pick email type and country
4. Click Generate - done!

## Built with
HTML, CSS, JavaScript - nothing fancy, just works.

## Installation

git clone https://github.com/kumoji-ops/Fake_Data_Generator.git
cd fake-data-generator
open index.html
